##Catering Management System

A Django-based web application for managing catering bookings, menu recommendations, customer feedback, and dashboard reporting.

## Features
- Home page with featured venues and menu items
- Online event booking
- Menu recommendations
- Feedback submission
- Simple sentiment analysis
- Dashboard reporting
- Django admin management

## Technologies Used
- Python
- Django
- SQLite
- HTML
- Bootstrap
- CSS

## Installation
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run migrations:
   python manage.py migrate
4. Start the server:
   python manage.py runserver

## Optional Demo Data
To load sample venues and menu items:
python manage.py seed_demo_data
